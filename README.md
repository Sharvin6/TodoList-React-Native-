# 📝 Todo List App

A mobile Todo application built with React Native using TypeScript and Javascript.  
This project focuses on learning and comparing different state management and data handling approaches, including Context API, Redux Toolkit, Thunk, Axios, and RTK Query.

---

## 🚀 Features

- Add, edit, and delete todos  
- Mark tasks as completed / undo  
- Priority selection (High / Normal)  
- Due date support  
- Overview dashboard (Total, Completed, Urgent, Normal)  
- Sorted todo list  

---

## 🧠 Key Learning

- Global state management using Context API  
- Structured state management using Redux Toolkit  
- Redux data flow:  
  Dispatch → Reducer → Store → UI Update  
- Handling async operations using Redux Thunk  
- API handling using Axios  
- Data fetching & caching using RTK Query  
- Using React Redux hooks (useSelector, useDispatch)  
- Debugging state using Reactotron  

---

## 🏗️ Tech Stack

- React Native  
- TypeScript  
- Redux Toolkit → build Redux
- React Redux → connect to UI
- Thunk → async logic 
- Axios  
- RTK Query  
- Reactotron
